# 📦 Supply Chain Performance Dashboard  

## 📌 Project Overview  
This project showcases an interactive **Supply Chain Performance Dashboard** built using **Power BI** and Excel datasets. The dashboard provides meaningful insights into sales, inventory, logistics, procurement, and order fulfillment to support data-driven business decisions.  

The project analyzes supply chain operations across multiple regions, countries, cities, and product categories using interactive visualizations and KPI metrics.

---

# 🎯 Objectives  
- Analyze overall supply chain performance  
- Track sales and profit growth trends  
- Monitor inventory and stock performance  
- Identify top-performing regions and categories  
- Improve operational and delivery insights  
- Enable better business decision-making through data visualization  

---

# 🛠️ Tools & Technologies  
- **Power BI** – Dashboard creation & visualization  
- **Microsoft Excel** – Data cleaning and preprocessing  
- **Python (Conceptual Understanding)**  
- **SQL (Basic Data Analysis Concepts)**

# 📊 Dashboard Features  

## 🔹 KPI Metrics  
- Total Sales Revenue  
- Total Orders  
- Average Order Value (AOV)  
- Forecast Accuracy  
- Inventory Turnover Ratio  
- Sales Growth vs Previous Year  
- Profit Growth vs Previous Year  

---

## 🔹 Visualizations Included  
- 📈 Sales Revenue by Year  
- 🌍 Sales by Country  
- 📦 Orders by Region  
- 💰 Profit by Region  
- 🏷️ Sub-category Cost Analysis  
- 🏙️ City-wise Sales Analysis  
- 📊 Sales & Price Trend Analysis  
- 🌐 Regional Revenue Distribution  

---

## 🔹 Interactive Filters  
- Region  
- Country  
- City  
- Product Category  
- Supplier  
- Delivery Mode  

---

# 📈 Key Insights  
- Asia generated the highest overall profit.  
- North America recorded the highest number of orders.  
- Forecast accuracy remained above 99%, showing effective demand planning.  
- Binders and Paper contributed significantly to total costs.  
- Sales performance showed steady growth across multiple years.
